# Tokslaw Portfolio

My personal portfolio website — a modern, responsive online presence showcasing my work, skills, and background as a full-stack software developer.

🌐 Live site: https://tokslaw7.github.io/portfolio/

## About

This repository contains the source code for my portfolio website. It highlights my experience in software development, cybersecurity, and product-focused engineering, with an emphasis on building clean, performant, and visually engaging web experiences.

## Features

- Responsive single-page portfolio
- Animated custom cursor and hover interactions
- Scroll reveal effects
- Animated stat counters
- Hero section with strong personal branding
- About, Projects, and Contact sections
- External links to blog and social profiles
- Downloadable resume/CV

## Tech Stack

- **HTML**
- **CSS**
- **JavaScript**

## Sections

- **Hero** — introduction
- **About** — background, experience, and core skills
- **Projects** — selected work and experiments
- **Contact** — ways to reach out and connect

## Getting Started

To run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/tokslaw7/portfolio.git
   ```

2. Open the project folder:
   ```bash
   cd portfolio
   ```

3. Open `index.html` in your browser, or serve it with a local dev server.

### Optional: Live Server
If you use VS Code, install the **Live Server** extension and launch the site from `index.html`.
